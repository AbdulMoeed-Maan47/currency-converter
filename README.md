I have started learning web development. This is a javascript based mini-project in the continuation of my learning.
I have implemented a web based currency converter using html, css, javascript.
I have used concepts like fetch API, promises, async-await, AJAX, DOM manipulation, event handling, arrays methods, web-based APIs and much more.
In this project, you can select the two currencies in which you want the conversion to take place.
The program uses an API to get the latest exchange rates between the two currencies from the internet and uses that to calculate and display the final amount.
It also uses an API to get and display the flags of the selected currencies on runtime.
It is a complete mini-project, that handles invalid inputs and shows correct outputs.
